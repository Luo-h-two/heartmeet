# 相亲交友平台 - 心遇 (HeartMeet)

一个简约温暖的相亲交友网站，包含完整的前端原型和后端系统。

## 技术栈

### 前端
- Vue 3 + Vite
- Pinia 状态管理
- Vue Router 路由
- Tailwind CSS 样式
- 原生 CSS 动画

### 后端
- Python FastAPI
- SQLAlchemy ORM
- SQLite 数据库
- JWT 鉴权
- Pydantic 数据校验

## 快速启动

### 后端
```bash
cd backend
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8000/docs 查看 Swagger 文档
```

### 前端
```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```
