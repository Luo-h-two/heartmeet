# 2. 页面结构说明

心遇是一个移动端优先的单页应用（SPA），所有页面通过 JavaScript 动态渲染，无需页面刷新。

## 页面总览

```
心遇 HeartMeet
├── 登录页       /login      - 手机号+密码登录
├── 注册页       /register   - 新用户注册（含密码强度检测）
├── 推荐页       /home       - 卡片式推荐（主要页面）
├── 发现页       /filter     - 筛选浏览所有用户
├── 消息页       /chat       - 聊天列表
├── 个人中心     /profile    - 查看/编辑个人资料
├── 行为分析     /analysis   - 用户行为数据报告
└── 用户详情     (Modal)     - 底部弹出式用户详情
```

## 各页面功能说明

### 1. 登录页
- 手机号 + 密码登录
- 表单校验：手机号格式、密码长度
- 登录成功后自动跳转到推荐页
- 登录状态持久化（localStorage）

### 2. 注册页
- 昵称、手机号、密码、性别
- **密码强度检测**：实时显示密码强度进度条
  - 红色：弱（仅数字或字母）
  - 黄色：中（数字+字母）
  - 绿色：强（数字+字母+特殊字符）
- 注册成功自动登录

### 3. 推荐页（首页）
- **卡片式展示**：大图 + 个人信息 + 标签 + 个人介绍
- **滑动交互**：
  - 触摸左滑 → 跳过
  - 触摸右滑 → 喜欢
- **按钮交互**：
  - ❤ 喜欢按钮（带脉冲动画）
  - ✕ 跳过按钮
  - 👋 打招呼按钮
- 点击卡片 → 弹出用户详情 Modal
- 互相喜欢 → 弹出「匹配成功」动画

### 4. 发现页
- 性别筛选（全部/男生/女生）
- 城市筛选（下拉选择）
- 年龄段筛选
- 标签筛选（多选）
- 网格布局展示用户卡片
- 点击进入用户详情

### 5. 消息页
- 聊天列表（展示有交互的用户）
- 显示最后一条消息预览
- 点击跳转用户详情

### 6. 个人中心
- 头像区域（可点击上传预览）
- **查看/编辑切换**：
  - 查看模式：信息行展示
  - 编辑模式：表单编辑 + 保存
- 择偶偏好设置（性别、年龄段）
- 兴趣标签管理（添加/展示）
- 行为分析入口
- 退出登录

### 7. 行为分析
- 行为总览：浏览数、喜欢数、跳过数、喜欢率
- 偏好标签 TOP5（进度条可视化）
- 偏好性别分布
- 年龄段偏好（柱状图可视化）

### 8. 用户详情（Modal）
- 底部弹出式设计
- 头像 + 基本信息
- 个人介绍
- 兴趣标签 + 性格标签
- 快速操作：喜欢/打招呼

## 全局功能

- **暗黑模式**：右上角月亮/太阳按钮切换
- **主题持久化**：刷新页面保持主题设置
- **Toast 提示**：操作结果反馈
- **移动端优先**：最大宽度 480px，平板/PC 居中显示
