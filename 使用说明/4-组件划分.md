# 4. 主要组件划分

## 前端组件树

```
App (#app)
├── Navbar (顶部导航栏)
│   ├── Brand (Logo + 标题)
│   ├── ThemeToggle (主题切换)
│   └── AuthButtons (登录/注册/退出)
├── PageContainer (页面容器 - 动态渲染)
│   ├── LoginForm (登录表单)
│   ├── RegisterForm (注册表单)
│   ├── RecommendCard (推荐卡片) *
│   ├── FilterBar (筛选栏)
│   ├── UserGrid (用户网格)
│   ├── ChatList (聊天列表)
│   ├── ProfileHeader (个人头部)
│   ├── ProfileInfo (基本信息 - 查看/编辑切换)
│   ├── PreferenceForm (择偶偏好表单)
│   ├── InterestTags (兴趣标签管理)
│   └── AnalysisReport (行为分析报告)
├── BottomNav (底部导航)
│   ├── NavItem (推荐)
│   ├── NavItem (发现)
│   ├── NavItem (消息 + 红点)
│   └── NavItem (我的)
├── Modal (通用模态框)
│   └── UserDetail (用户详情)
├── MatchDialog (匹配成功弹窗)
└── Toast (提示消息)
```

## 核心组件说明

### 1. RecommendCard（推荐卡片）
```
推荐卡片
├── CardImage (用户头像区域)
│   ├── AvatarEmoji (头像表情)
│   ├── GradientOverlay (底部渐变遮罩)
│   └── CardInfo (姓名/年龄/城市/职业)
├── TagList (标签列表)
│   ├── Tag (普通标签)
│   └── HighlightTag (共同兴趣高亮)
├── RecommendReason (推荐理由)
└── ActionButtons (操作按钮)
    ├── SkipBtn (跳过)
    ├── GreetBtn (打招呼)
    └── LikeBtn (喜欢 - 带脉冲动画)
```

**交互逻辑**：
- 触摸滑动检测（左滑跳过/右滑喜欢）
- 点击卡片查看详情
- 三个操作按钮各司其职

### 2. ProfileInfo（个人资料 - 双模式）
```
ProfileInfo
├── ViewMode (查看模式)
│   ├── InfoRow (昵称)
│   ├── InfoRow (年龄)
│   ├── InfoRow (性别)
│   ├── InfoRow (城市)
│   ├── InfoRow (职业)
│   └── BioText (个人介绍)
└── EditMode (编辑模式)
    ├── FormInput (昵称)
    ├── FormInput (年龄)
    ├── FormSelect (性别)
    ├── FormInput (城市)
    ├── FormInput (职业)
    ├── FormTextarea (个人介绍)
    └── SaveBtn (保存)
```

### 3. AnalysisReport（行为分析）
```
AnalysisReport
├── StatsCards (统计卡片)
│   ├── TotalViews (总浏览)
│   ├── TotalLikes (总喜欢)
│   ├── TotalSkips (总跳过)
│   └── LikeRate (喜欢率)
├── TagChart (标签偏好 - 进度条)
├── GenderChart (性别偏好 - 饼图模拟)
└── AgeChart (年龄偏好 - 柱状图模拟)
```

### 4. UserDetail（用户详情 Modal）
```
UserDetail
├── ModalHeader (标题 + 关闭)
├── Avatar (头像表情)
├── UserBasicInfo (基本信息)
├── Bio (个人介绍)
├── InterestTags (兴趣标签)
├── PersonalityTags (性格标签)
└── ActionButtons (喜欢/打招呼)
```

## 状态管理

```
Store (全局状态)
├── token          (JWT Token - localStorage 持久化)
├── user           (用户信息 - localStorage 持久化)
├── theme          (主题 - localStorage 持久化)
├── currentPage    (当前页面)
└── 方法
    ├── setToken()     - 设置 Token
    ├── setUser()      - 设置用户信息
    ├── setTheme()     - 设置主题
    ├── isLoggedIn()   - 检查登录状态
    └── logout()       - 退出登录
```
