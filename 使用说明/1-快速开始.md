# 1. 快速开始

## 环境要求

- **Python** 3.9+
- **Node.js** 16+ (仅用于可选的前端开发服务器)
- 操作系统：Windows / macOS / Linux

## 启动后端

```bash
# 1. 进入后端目录
cd heart-meet/backend

# 2. 创建虚拟环境（推荐）
python -m venv venv

# 3. 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 启动后端服务
python main.py
```

后端启动后：
- API 地址：`http://localhost:8000`
- Swagger 文档：`http://localhost:8000/docs`
- 数据库自动初始化，包含 12 个模拟用户

## 启动前端

### 方式一：直接用浏览器打开（推荐，最简单）

```bash
# 用浏览器直接打开 index.html
# Windows:
start frontend/index.html

# macOS:
open frontend/index.html
```

由于前端使用了 `fetch` 调用 `localhost:8000`，建议使用本地服务器：

```bash
# 进入前端目录
cd heart-meet/frontend

# 使用 Python 启动静态服务器
python -m http.server 5173

# 浏览器访问 http://localhost:5173
```

### 方式二：使用 Vite（可选）

```bash
cd heart-meet/frontend
npx vite . --port 5173
```
